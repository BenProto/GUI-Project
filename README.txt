This is a weather app that shows the current weather along with a 10-hour
forecast. In addition to this, it will show the weather at specific times
which the user inputs as well as suggestions for what the user should wear/bring
based on the upcoming weather.

In order to run this app, follow the below instructions:
1. Download this repository onto your local machine.
2. "nom install" to install general dependencies.
3. "npm install react-router-dom" to install the necessary routing dependencies.
4. "npm run dev" to run the web application.
